# VolunteerHub

VolunteerHub is a web-based application designed to manage volunteer events, track registered volunteers, and share resources. The project provides a modern design for a user-friendly experience.

**Features**
- **Home**: General introduction to the project
- **Registered Volunteers**: List of registered volunteers
- **Active Events**: Tracking of ongoing events
- **Resources**: Useful materials for volunteers
- **Partners**: Organizations collaborating with the project
- **Contact**: Contact form for communication
- **Dark Mode**: Switch themes using the Theme Toggle button

 **Technologies Used**
- **HTML5** – page structure
- **CSS3** – styling, responsive layout, and gradient backgrounds
- **JavaScript** – interactive elements and dynamic behavior
- **PHP** – server-side scripting and backend logic
- **SQL** – database design and queries for storing volunteers, events, and resources
